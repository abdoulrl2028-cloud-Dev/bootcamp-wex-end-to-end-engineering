# Roteiro de Teste Manual

## Caso de Teste: Login do Usuário

**Pré-condição:** Usuário cadastrado no sistema

**Passos:**
1. Abrir o sistema
2. Informar usuário e senha válidos
3. Clicar em "Entrar"

**Resultado Esperado:**
- Usuário deve ser autenticado e direcionado para a tela principal

---

## Caso de Teste: Falha no Login

**Pré-condição:** Usuário cadastrado no sistema

**Passos:**
1. Abrir o sistema
2. Informar usuário ou senha inválidos
3. Clicar em "Entrar"

**Resultado Esperado:**
- Mensagem de erro exibida: "Usuário ou senha inválidos"
