# AWS Cloud - Fundamentos e Práticas

Este módulo aborda os principais serviços da AWS (Amazon Web Services) com exemplos práticos e projetos completos.

## Estrutura do Módulo

- **ec2/**: Instâncias EC2, configurações e automação
- **s3/**: Armazenamento em S3, upload/download e políticas
- **redes/**: VPC, subnets, security groups e load balancers
- **banco-dados/**: RDS, DynamoDB e gerenciamento de dados
- **projeto-custos/**: Projeto prático de otimização de custos

## Pré-requisitos

- Conta AWS ativa
- AWS CLI configurado
- Credenciais AWS configuradas localmente
- Python 3.8+
- Boto3 instalado

## Instalação de Dependências

```bash
pip install boto3 awscli
```
