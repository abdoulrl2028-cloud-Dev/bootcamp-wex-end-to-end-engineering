# 4 - Bancos de Dados

Este módulo cobre os fundamentos de bancos de dados relacionais e não-relacionais.

## Estrutura

### SQL Server
- **tabelas/**: Criação e estrutura de tabelas SQL
- **manipulacao/**: Operações CRUD (Create, Read, Update, Delete)
- **projeto-consultas/**: Projeto prático com consultas avançadas

### MongoDB
- Operações com documentos NoSQL
- Queries e agregações

## Conteúdo

- Conceitos de bancos de dados
- DDL (Data Definition Language)
- DML (Data Manipulation Language)
- Relações entre tabelas
- Queries e índices
- NoSQL com MongoDB
