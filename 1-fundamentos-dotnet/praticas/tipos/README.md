# Prática: Tipos e Conversões

Este exercício demonstra tipos primitivos, conversões e formatação.

Como executar:

```bash
cd 1-fundamentos-dotnet/praticas/tipos
dotnet run
```

Saída esperada (exemplo):

```
int: 42
double: 3.1415
string: exemplo
bool: True
Conversão double -> int: 3
Valor formatado: 3.14
```
