# Estruturas de Repetição em C#

Este projeto demonstra as principais estruturas de repetição disponíveis em C#.

## Conteúdo

### 1. **FOR**
- Repetição com contador
- Tabuadas
- FOR aninhado

### 2. **WHILE**
- Repetição com condição
- Menus interativos
- Validação de entrada

### 3. **DO-WHILE**
- Sempre executa pelo menos uma vez
- Menus com validação

### 4. **FOREACH**
- Iteração sobre arrays
- Iteração sobre coleções

### 5. **Controle de Fluxo**
- `break`: Encerra o loop
- `continue`: Pula para próxima iteração

### 6. **Aplicações Práticas**
- Cálculo de soma
- Cálculo de fatorial
- Matrizes

## Como executar

```bash
dotnet run
```

## Conceitos-chave

- **FOR**: Use quando souber quantas vezes repetir
- **WHILE**: Use quando a condição determina o fim
- **DO-WHILE**: Use quando precisa executar pelo menos uma vez
- **FOREACH**: Use para iterar sobre coleções
