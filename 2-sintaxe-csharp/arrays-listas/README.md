# Arrays e Listas em C#

Este projeto demonstra o uso de arrays e listas, incluindo operações comuns e LINQ.

## Conteúdo

### 1. **Arrays**
- Arrays unidimensionais
- Inicialização de arrays
- Diferentes tipos de dados

### 2. **Arrays Multidimensionais**
- Matrizes (2D)
- Jagged Arrays (arrays de arrays)

### 3. **Listas (List<T>)**
- Criação de listas dinâmicas
- Adição de elementos (`Add`)
- Inserção (`Insert`)
- Remoção (`Remove`, `RemoveAt`)
- Busca (`Contains`, `IndexOf`)

### 4. **Ordenação**
- `Sort()`: Ordena ascendente
- `Reverse()`: Inverte a ordem

### 5. **LINQ**
- `Where()`: Filtragem
- `Select()`: Transformação
- `FirstOrDefault()`: Encontra primeiro elemento
- `Any()`: Verifica existência
- `Count()`: Conta elementos com condição

### 6. **Operações Úteis**
- `Average()`: Média
- `Max()`: Máximo
- `Min()`: Mínimo
- `Clear()`: Limpa lista

## Como executar

```bash
dotnet run
```

## Quando usar?

- **Array**: Quando sabe o tamanho fixo
- **List<T>**: Quando o tamanho é dinâmico
- **Array Multidimensional**: Para dados em matriz
- **Jagged Array**: Para matrizes com linhas de tamanhos diferentes
